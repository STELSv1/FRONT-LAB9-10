# 🚂 WEB-project — Ukrainian Railway Tickets

> **Лабораторна робота 9–10** |

---

## 🛠 Технології

| Технологія | Призначення |
|---|---|
| React 18 | UI-бібліотека |
| Vite | Збірник / dev-сервер |
| React Router v6 | Клієнтська маршрутизація |
| Context API + useReducer | Глобальний стан |
| CSS Modules | Стилізація компонентів |

---

## 🚀 Запуск проекту

```bash
# Встановлення залежностей
npm install

# Запуск dev-сервера
npm run dev

# Збірка для продакшену
npm run build
```

---

## 📁 Структура проекту

```
src/
├── components/        # Перевикористовувані UI компоненти
│   ├── Navbar/        # Навігаційна панель
│   ├── TrainCard/     # Картка рейсу
│   ├── SearchForm/    # Форма пошуку
│   └── BookingForm/   # Форма бронювання
│
├── context/           # Context API — глобальний стан
│   └── TicketContext.jsx
│
├── pages/             # Сторінки застосунку
│   ├── HomePage/      # Головна — пошук рейсів
│   ├── TrainsPage/    # Список доступних рейсів
│   ├── BookingPage/   # Бронювання місця
│   └── ConfirmationPage/  # Підтвердження квитка
│
├── routes/            # Конфігурація маршрутизації
├── data/              # Mock-дані (рейси, міста)
├── utils/             # Допоміжні функції (дати, фільтри)
└── styles/            # Глобальні стилі
```

---

## ✨ Функціонал

- 🔍 **Пошук рейсів** — за містом відправлення, призначення та датою
- 🚆 **Список потягів** — перегляд доступних рейсів із фільтрацією
- 🎫 **Бронювання** — вибір місця, введення даних пасажира
- ✅ **Підтвердження** — сторінка з деталями квитка та номером бронювання
- 🌐 **SPA-маршрутизація** — React Router з захищеними маршрутами
